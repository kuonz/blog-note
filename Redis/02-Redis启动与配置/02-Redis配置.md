| 参数      | 说明                                                   | 默认值 |
| --------- | ------------------------------------------------------ | ------ |
| daemonize | 是否以守护进程方式启动                                 | on     |
| port      | redis占用端口                                          | 6379   |
| logfile   | redis系统日志名称                                      |        |
| dir       | redis工作目录：存放redis工作时的各种信息，如数据和日志 |        |
